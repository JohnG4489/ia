# Research

Ce dossier contient tous les documents de recherche et articles académiques.

## Contenu

- **Papers** : Articles de recherche annotés
- **Surveys** : États de l'art par domaine
- **Experiments** : Reproductions d'expériences
- **Notes** : Prises de notes de lecture
- **Presentations** : Slides de conférences
- **Benchmarks** : Comparaisons de modèles

## Structure

```
research/
├── papers/
├── surveys/
├── experiments/
├── notes/
├── presentations/
├── benchmarks/
└── bibliography/
```

## Format

- Documents en Markdown
- Notebooks Jupyter pour les expériences
- PDF pour les articles originaux
- Bibliographie en BibTeX
